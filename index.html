Debido a los problemas de la "política del mismo origen", decidí usar dos alternarivas: una usando una solicitud http para leer el archivo json y analizar el array de objetos y la otra con el array de objetos cargados normalmente en la memoria como un archivo JavaScript. Ustedes pueden probar la versión que deseen (si no funciona la primera versión que se lee del archivo json, es solo el mismo problema de política). La aplicación es la misma en ambas versiones, la única diferencia es la forma en que se cargan los datos del los usuarios.
